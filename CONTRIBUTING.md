# Contributing to InfiShark Docs

Thanks for taking the time to contribute! Even small changes help us improve.

## What You Can Help With

- Fixing typos or formatting
- Improving clarity or examples
- Adding missing steps or edge cases
- Updating content for new features

## Setup

1. Fork this repository
2. Clone your fork
3. Run `npm install`
4. Run `npm run start` to see the site locally

## Guidelines

- Keep pull requests small and focused.
- Use clear & concise language.
- Keep the tone consistent with the rest of the docs.
- No AI-generated content unless manually reviewed.

## Submitting a Pull Request

1. Create a new branch from `main`
2. Make your changes
3. Push and open a pull request
