The BLEShark Nano includes two simple but useful utilities: a countdown timer and a stopwatch.  
Both use the same time format: <code>MM:SS:CC</code> (minutes, seconds, centiseconds).

### Using the Timer App

<ul className="step-list">
  <li>Open the <strong>Timer</strong> app from the Apps menu.</li>
  <li>The screen will show <code>00:00:00</code>. The currently selected unit appears in a larger font.</li>
  <li>Press <strong>[S]</strong> to move between minutes, seconds, and centiseconds.</li>
  <li>Use <strong>[L]</strong> to decrease the selected value and <strong>[R]</strong> to increase it.</li>
  <li>When you reach the last unit and press <strong>[S]</strong>, the timer will start.</li>
  <li>Once the countdown reaches zero, the screen will flash repeatedly.</li>
</ul>

### Using the Stopwatch App

<ul className="step-list">
  <li>Open the <strong>Stopwatch</strong> app from the Apps menu.</li>
  <li>The screen will show <code>00:00:00</code>.</li>
  <li>Press <strong>[S]</strong> to start or pause the timer.</li>
  <li>Press <strong>[L]</strong> to reset the stopwatch to zero.</li>
</ul>

:::info
The BLEShark Nano uses a flashing screen to indicate that the timer has ended - there is no buzzer or vibration.
:::
