The TxtViewer app lets you read plain text files directly on your BLEShark Nano’s OLED screen. Great for study notes, documentation, or stealthy cheat sheets during tests. It supports smooth vertical scrolling, word wrapping, and an on-screen scrollbar.

### How to Use TxtViewer

<ul className="step-list">
  <li>Open the <strong>TxtViewer</strong> app from the Apps menu.</li>
  <li>Use <strong>[L]</strong> to scroll up and <strong>[R]</strong> to scroll down.</li>
  <li>Hold <strong>[S]</strong> to exit the app at any time.</li>
</ul>

When you first launch the app, a default placeholder file may be shown.  
If no file has been uploaded, you'll see instructions on how to add your own.

### How to Upload a Text File

<ul className="step-list">
  <li>Enter <strong>Extra Settings Mode</strong>.</li>
  <li>Connect to the BLEShark Nano’s Wi-Fi hotspot.</li>
  <li>Open the captive portal in your browser.</li>
  <li>Go to <strong>App Settings</strong> and click <strong>Upload TxtViewer File</strong>.</li>
  <li>Upload your UTF-8 text file, then hit <strong>Apply Changes</strong>.</li>
</ul>

Your file will now be readable in the TxtViewer app.

:::info[File Format and Limitations]
- TxtViewer supports standard <strong>UTF-8</strong> encoded text files.  
- Maximum file size: <strong>25KB</strong> (~25,000 characters).  
:::

This limitation ensures optimal performance and compatibility with the BLEShark Nano’s screen and storage system.
