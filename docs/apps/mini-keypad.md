---
title: Mini Keypad
---

The Mini Keypad app turns your BLEShark Nano into a tiny customizable shortcut pad.  
You can assign up to 8 different key combinations using the device’s buttons.  
### How to Use the Mini Keypad

<ul className="step-list">
  <li>Open the <strong>Mini Keypad</strong> app from the Apps menu.</li>
  <li>Connect to the BLEShark Nano over Bluetooth.</li>
  <li>Press a button or combo to trigger the assigned command.</li>
</ul>

### Button Combinations

View the full shortcut layout in the [Shortcut Guide](/docs/misc/shortcut-guide).

### How to Edit Keypad Commands

<ul className="step-list">
  <li>Enter <strong>Extra Settings Mode</strong> on your BLEShark Nano.</li>
  <li>Connect to the BLEShark Wi-Fi network when it appears.</li>
  <li>Open the captive portal in your browser.</li>
  <li>Go to <strong>App Settings</strong> and select <strong>Keypad Settings</strong>.</li>
  <li>Enter your desired command (example: <code>ctrl+alt+delete</code>).</li>
</ul>

:::tip
Use the correct format: <code>key+key+key</code> with no spaces.
:::

### Example Commands

- <code>ctrl+c</code>  
- <code>ctrl+v</code>  
- <code>alt+tab</code>  
- <code>fn+alt+f4</code>

### Supported Keys

<pre>
ctrl, shift, alt, fn, esc, enter, f1-f12, tab, space, backspace, delete,
home, end, pageup, pagedown, insert, arrowup, arrowdown, arrowleft, arrowright,
capslock, numlock, scrolllock, meta, playpause, stop, nexttrack, prevtrack,
volup, voldown, printscreen
</pre>

- <strong>Meta</strong> represents the Windows key on Windows or the ⌘ key on macOS.
